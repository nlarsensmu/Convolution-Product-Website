Our Mediums

We provide 3 different contexts a node library without threading, an electron app, and a library with theading. The library was created using electron to compile C++ code into a JavaScript WASM library.

  • 01 Running in a browser

    A demo website is provided to run your images. To gather data on how fast the library runs compared to a native app, we ran a node program that consumes the library and runs a Convolution on an input image. The Native was always faster but you can see on the means chart, WASM was only marginally slower. Images were run on a 1920 × 1080
